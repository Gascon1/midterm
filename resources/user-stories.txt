Requirements/Features :

- The user simply has to add the name of the thing, and it gets put into the correct list.

- Each todo created should be categorized as one of:

    Film / Series (To watch)
    Restaurants, cafes, etc. (To eat)
    Books (To read)
    Products (To buy)
    * Not categorized (special category for todo items not categorized)

- Users should be able to change a category of an item in case it was mis-categorized or
  could not be categorized at all.

- Users should be able to register, log in, log out and update their profile.

STRETCH FEATURES:

- If the todo created is in two categories, prompt the user to choose the right one based on his
  intent or leave the todo in both categories

- Users can share a specific category through a link to our app. Once the app the new User can
  see the shared list and is invited to create an account to save the shared list in his own lists

- Users can enter a verb as keyword to specify the category. Example: Read Harry Potter would only
  add the todo to the Books category

- Users can create a new category
